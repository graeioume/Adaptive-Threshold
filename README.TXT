This code was written in Xcode, so the instructions to open and run should be as follows:
https://stackoverflow.com/a/5156193/5095372

The image files must be contained in the folder that contains the Executable.
The Debug build folder as it should appear is provided, containing the executable, input images, and output images. 
The build folder should be set to the path:
	…/Adaptive Threshold/DerivedData/Adaptive Threshold/Build/Products/Debug/Carnegie Robotics Coding Challenge
If not, set the build settings via File-> Project Settings, and choose “Project-relative Location” DerivedData
